# Traders Edge Network — Portal Tutorials

A simple tutorial site for sharing walkthrough videos that help members learn how to use the [Traders Edge Network member portal](https://hub.tradersedgenetwork.com).

## Videos

1. **The Foundation & Membership** — Portal orientation and membership overview
2. **PaperMoney & Placing a Trade** — Practice trading walkthrough

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deployment

The site deploys automatically on **Vercel** when you push to `main`.

Live site: https://traders-edge-network.vercel.app

GitHub Pages is also configured via Actions if you prefer that host — enable it under **Settings → Pages → GitHub Actions**.
